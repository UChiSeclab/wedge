```python
import os
import random

def generate_test_case(n, k, max_value=104):
    """
    Generates a single test case.
    n: number of integers in the sequence.
    k: number of bits in which integers in k-interesting pair should differ.
    max_value: maximum value an integer in the sequence can take.
    """
    sequence = [random.randint(0, max_value) for _ in range(n)]
    return n, k, sequence

def write_test_case(directory, index, test_case):
    """
    Writes the test case to a file in the specified directory.
    directory: output directory for the test case file.
    index: index for naming the test case file.
    test_case: tuple containing the test case data (n, k, sequence).
    """
    n, k, sequence = test_case
    with open(os.path.join(directory, f'test_{index:02d}.in'), 'w') as file:
        file.write(f"{n} {k}\n")
        file.write(" ".join(map(str, sequence)) + "\n")

def generate_test_cases(directory, num_cases=10, max_n=100000, max_value=104, max_k=14):
    """
    Generates a specified number of test cases and writes them to the directory.
    directory: output directory for the test case files.
    num_cases: number of test cases to generate.
    max_n: maximum size of the sequence.
    max_value: maximum value an integer in the sequence can take.
    max_k: maximum value of k.
    """
    if not os.path.exists(directory):
        os.makedirs(directory)
    
    for i in range(1, num_cases + 1):
        n = random.randint(2, max_n)
        k = random.randint(0, max_k)
        test_case = generate_test_case(n, k, max_value)
        write_test_case(directory, i, test_case)

if __name__ == "__main__":
    import argparse
    
    parser = argparse.ArgumentParser(description="Generate test cases for the k-interesting pairs problem.")
    parser.add_argument('directory', type=str, help="Output directory for the test cases.")
    parser.add_argument('--num_cases', type=int, default=10, help="Number of test cases to generate. Default is 10.")
    parser.add_argument('--max_n', type=int, default=100000, help="Maximum size of the sequence. Default is 100000.")
    parser.add_argument('--max_value', type=int, default=104, help="Maximum value an integer in the sequence can take. Default is 104.")
    parser.add_argument('--max_k', type=int, default=14, help="Maximum value of k. Default is 14.")
    
    args = parser.parse_args()
    
    generate_test_cases(args.directory, args.num_cases, args.max_n, args.max_value, args.max_k)
```

### Explanation:
1. **generate_test_case**: Generates a single test case with `n` random integers (each between 0 and `max_value`) and a random `k` between 0 and `max_k`.

2. **write_test_case**: Writes a test case to a file named `test_{index:02d}.in` in the specified directory.

3. **generate_test_cases**: Generates a given number of test cases and writes them to the specified directory.

4. **main function**: Uses the argparse library to handle command-line arguments for output directory, number of test cases, maximum size of sequence (`n`), maximum value of integers in sequence, and maximum value of `k`.

This script is intended to be run using the command line. It reads parameters to determine how many test cases to generate and where to store them. The generated test cases will help in testing the performance and correctness of the provided solutions under heavy load conditions.
