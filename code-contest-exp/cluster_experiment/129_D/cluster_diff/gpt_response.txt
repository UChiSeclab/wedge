```python
import os
import sys
import random
import string

def generate_test_cases(num_cases, max_string_length, max_k_value):
    test_cases = []

    for i in range(num_cases):
        # Generate a random string of maximum length
        length = max_string_length
        s = ''.join(random.choice(string.ascii_lowercase) for _ in range(length))
        
        # Generate maximum k value
        k = max_k_value

        test_cases.append((s, k))

    return test_cases

def write_test_cases_to_directory(test_cases, directory):
    if not os.path.exists(directory):
        os.makedirs(directory)
        
    for i, (s, k) in enumerate(test_cases):
        file_name = os.path.join(directory, f'test_{i+1:02d}.in')
        with open(file_name, 'w') as f:
            f.write(f"{s}\n")
            f.write(f"{k}\n")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Please provide the directory to save test cases.")
        sys.exit(1)
    
    directory = sys.argv[1]
    num_cases = 20
    max_string_length = 100000  # Maximum allowed length of string
    max_k_value = 100000        # Maximum allowed value of k

    test_cases = generate_test_cases(num_cases, max_string_length, max_k_value)
    write_test_cases_to_directory(test_cases, directory)
    print(f"Generated {num_cases} test cases and saved to {directory}")
```

### Explanation:
1. **Importing Necessary Modules**:
   - `os` and `sys` for directory and argument handling.
   - `random` for generating random characters.
   - `string` for accessing alphabetic characters.

2. **`generate_test_cases` Function**:
   - Generates a specified number of test cases.
   - Each test case consists of a string `s` of maximum length (`max_string_length`) filled with random lowercase letters, and the largest possible value of `k` within the problem's constraints (`max_k_value`).

3. **`write_test_cases_to_directory` Function**:
   - Writes generated test cases to the specified directory with sequentially numbered filenames (`test_01.in`, `test_02.in`, ..., `test_num_cases.in`).
   - Creates the directory if it doesnâ€™t exist.

4. **Main Script Execution**:
   - Takes the target directory from the script's argument to save the test cases.
   - Defines the number of test cases, maximum string length, and maximum k value based on problem constraints.
   - Calls the function to generate and write test cases to the specified directory.

### Usage:
Run the script from the command line and provide the directory as an argument where the test cases should be saved, for example:

```sh
python generate_test_cases.py ./test_directory
```

This command will generate 20 test cases with large inputs to stress test the solution and save them in the specified directory. Adjust `num_cases`, `max_string_length`, and `max_k_value` to match your requirements.
